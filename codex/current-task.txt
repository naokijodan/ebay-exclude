~/Desktop/ebay-exclude/src/lib/auth.ts のrefreshToken関数を修正してください。

## 問題
トークンリフレッシュ時に `scope` パラメータを送っているが、eBayのOAuth APIは
リフレッシュトークンが発行された時のスコープと一致しないとエラーになる。
RAKUDAプロジェクトのworkerでは `scope` を送っていない。

## 修正内容
`refreshToken()` 関数内の `URLSearchParams` から `scope` 行を削除する。

修正前:
```typescript
  const body = new URLSearchParams({
    grant_type: 'refresh_token',
    refresh_token: refreshToken,
    scope: 'https://api.ebay.com/oauth/api_scope/sell.account',
  });
```

修正後:
```typescript
  const body = new URLSearchParams({
    grant_type: 'refresh_token',
    refresh_token: refreshToken,
  });
```

~/Desktop/ebay-exclude/src/lib/auth.ts のみ修正

- scopeの行だけ削除してください
