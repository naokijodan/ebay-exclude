## 新しいUXフロー

起動すると、まずウェルカムメッセージと最初の質問が表示される：

```
╔══════════════════════════════════════╗
║   ebay-exclude - 発送除外国管理     ║
╚══════════════════════════════════════╝

こんにちは！eBayの発送除外国を管理します。

? 何をしますか？
  1) 現在の設定を確認する
  2) 除外国を変更する
  3) 終了

>
```

### 「1) 現在の設定を確認する」を選んだ場合:

```
ポリシー一覧を取得中...

[ポリシー一覧が表示される]

? 次にどうしますか？
  1) 特定のポリシーの除外設定をCSVに出力する
  2) メインメニューに戻る

>
```

CSVに出力を選んだ場合:
```
? 出力ファイル名は？（Enterでexport.csv）:

export.csv に保存しました！

? 次にどうしますか？
  1) このCSVを編集してeBayに反映する手順を見る
  2) メインメニューに戻る
```

「手順を見る」を選んだ場合:
```
━━━ CSVを編集してeBayに反映する手順 ━━━

  Step 1: export.csv をExcelやテキストエディタで開く
  Step 2: action列を変更する
          - exclude = 除外する
          - include = 除外しない（許可する）
  Step 3: 保存したら、このツールに戻って
          「2) 除外国を変更する」を選んでください

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

? メインメニューに戻りますか？(Enter)
```

### 「2) 除外国を変更する」を選んだ場合:

```
? CSVファイルはもう準備できていますか？
  1) はい、CSVがあります
  2) いいえ、まずテンプレートCSVを作成する
  3) メインメニューに戻る

>
```

「テンプレートCSVを作成する」を選んだ場合:
```
? 出力ファイル名は？（Enterでexclusions.csv）:

exclusions.csv を作成しました！

━━━ 次のステップ ━━━
  1. exclusions.csv をExcelやテキストエディタで開く
  2. 除外したい地域/国のaction列を「exclude」に変更
  3. 許可したい国のaction列を「include」に変更
  4. 保存して、再度このウィザードを起動してください
━━━━━━━━━━━━━━━━━━━━━━━

? メインメニューに戻りますか？(Enter)
```

「CSVがあります」を選んだ場合:
```
? CSVファイルのパスを入力してください
  （ファイルをここにドラッグ＆ドロップもできます）

>

変更内容をプレビューしています...

[diffの結果が表示される]

? この変更をeBayに適用しますか？
  1) はい、適用する
  2) いいえ、やめる

>
```

「はい」を選んだ場合:
```
適用中...

✔ 完了しました！変更がeBayに反映されました。

? メインメニューに戻りますか？(Enter)
```

## 実装の注意点

- readline を使用（追加パッケージ不要）
- chalk@4（CommonJS）を使用
- 子プロセスで既存コマンドを実行（runCommand関数は現在のものを流用）
- fileExists関数も流用
- 質問のプロンプトは `> ` にする
- 番号入力は 1, 2, 3 等のシンプルな数字
- 無効な入力の場合は「1-3の番号を入力してください」等のエラーメッセージ
- Enter だけ押した場合のデフォルト値対応
- ドラッグ＆ドロップされたパスは前後の空白をtrimする
- 各操作後は必ず「メインメニューに戻る」か「終了」の選択肢を出す
- runWizard() のexport、関数シグネチャは変えない
